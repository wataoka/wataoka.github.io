# 綿岡晃輝

**Responsible AIチーム, SB Intuitions / ソフトバンク**

![Profile Image](wataoka2025.jpg)

## 概要

SB Intuitions Responsible AIチーム リーダー / Chief Research Engineer。LLMやVLMにおける安全性評価や訓練、レッドチーミングやガードレールなどの**AIセーフティ**に取り組んでいます。研究分野としては、LLM-as-a-Judgeにおけるバイアス、ジェイルブレイク攻撃、LLMのフィンガープリンティング、機械学習の公平性、プライバシー保護計算などです。
