言語スキル

python ★★★★★

6年間ほぼ毎日使い続けた言語です。機械学習にもWeb開発にも使用し、文句無しに使えます。

HTML/CSS ★★★★★

適切にレスポンシブ対応したサイトを作れます。当サイト(wataoka.info)もレスポンシブ対応しています。

JavaScript ★★★

趣味のWeb開発でいじる程度です。

Typescript ★★

クックパッドインターンの際に本腰を入れて勉強しました。Webアプリ「人間選別」のサーバーはTypescriptを使用しました。最近使っていないです。

C/C++ ★★★

大学の授業で4年間習ってきました。ポインタ, 名前空間, STLなど必須事項は抑えています。授業でリードソロモン符号の設計をCで行いました。また、競プロでたまに使います。

Java/Scala ★★★

制約充足問題のソルバーのための構文解析器とパーサーと処理系をscalaで書いたことがあります。Sprint bootをちょっとだけ触ったことあります。

R ★★★★

他人が書いたコードを頻繁に読むことがあります。もちろんデータセットの用意からモデルの評価までを適切に書くこともできます。

Ruby ★★★

クックパッドインターンの際に頻繁に触りました。railsで簡単なブログサイトを作ったことがあります。

PHP ★★

Laravelを用いたバックエンドAPIを作ったことがあります。

Swift ★★★★

大学2回生までiOSアプリ開発をしていました。また、クックパッドインターンの際にcookpad liveというアプリのプロジェクトに配属され、swiftを使用しました。

Go ★

Tour of Goをしたことがあるだけです。


Webフレームワーク

Django ★★★★

使用頻度の高いフレームワークです。個人的な全体の開発経験もバックエンド側の開発経験もあります。しっかりとしたプロジェクト管理もできていると思います。

Flask ★★★★

このサイトもflaskを使用しており、継続的に更新を続けていますので、頻繁に触れるフレームワークです。

Laravel ★★

友人との共同開発でバックエンドを担当することになった時に、APIを設計するために使用しました。

Ruby on rails ★★

練習のために簡単なブログサイトを作っただけで、慣れているとは言えません。

Spring boot ★★

練習のために簡単なTODOアプリを作ったことがあるだけで、慣れているとは言えません。

RxSwift ★★★

クックパッドインターンの際に使用しましたが最近触っていないです。

その他 bootstrapやjQueryなどのよくあるライブラリ ★★★★

フロントエンドは勉強が泥臭いのであまり好きではありませんが、何度も使ってきているので、嫌でも適切に使えるようになったと思います。特にbootstrapはほとんど全ての開発に使用しているので、とても慣れ親しんでいます


機械学習フレームワーク

Pytorch ★★★★★

毎日触っているフレームワークです。並列化も含め問題なく使えます。フレームワーク自体の勉強も深く行いましたので、多少は内部構造も知っているつもりです。

Tensorflow ★★★★

修士の研究でPytorchと併用して使用しており、ほとんど毎日触っています。学部の時はKerasの限界が見えた時にTensorflowを使用してきました。Tensorflowで書かれた他人のコードを読むのが苦手なので星4つとしましたが、Tensorflowがそもそも読みにくいのだと信じています。(1.x系の話)

Keras ★★★★★

学部3~4回生の2年間使用してきたフレームワークです。Kerasの限界が見えるほど自分で設計してきました。

sklearn ★★★★

もちろん御用達ですが、体系的に知っているわけではないので4としました。

Chainer ★

Udemyの現場で使えるChainerによるディープラーニング入門を受講したことがあるだけです。

Optuna ★★★★

ハイパーパラメータ探索はほとんどOptunaで行います。一時期、hyperasというハイパーパラメータ探索ライブラリを使用していましたが、現在は完全にOptunaに移行しています。

その他Numpy, Pandasなどの必須ライブラリ ★★★★★

呼吸をするように使用してきました。


その他

Git ★★★★★

研究でもWeb開発でも必ず使用しています。一人で行うプロジェクトでもきっちりとした管理を心がけてきましたので、適切に使えていると思います。

Docker/Vagrant ★★★

何度も必要にかられ使用してきていますので基本的なシステムや使用方法は理解していますが、若干苦手意識があります。

SQL ★★★

MySQL, SQLite, PostgreSQLを使用したことがあります。ISUCONに出場時に最適化の勉強も頑張りましたが、胸を張って得意だとは言えません。

英語 ★★★★

機械学習の論文を毎日読んでいますので、リーディングは全く問題ありません。リスニングやスピーキングはDMM英会話を半年ほど(?)受けました。日常会話なら支障なくできますが、スラスラと話せる自信はありません。TOEICは730点です。(2018年5月)

数学 ★★★★

この世で最も好きなものは数学です。数3, 線形代数, 解析, 複素関数, 確率統計, 集合論, 情報数学など高校数学と情報系の大学数学の基本はしっかりと抑えています。他には数論, 情報幾何学, 代数幾何学, 多様体, グレブナ基底などを勉強していますが, どれも何度も挫折しています。数学検定は準1級です。(2014年10月くらい)
